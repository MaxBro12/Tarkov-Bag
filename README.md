# TarkovBag

Приложение для координации по поиску лута в Escape From Tarkov. 

## Установка

### Ручная - Тестовая стреда

Установка

```shell
cd backend
uv sync
cd ../frontend
npm install
```

Запуск бэкенда

```shell
cd backend
. ./.venv/bin/activate
python src
```

Запуск фронтенда

```shell
cd frontend
npm run dev
```
